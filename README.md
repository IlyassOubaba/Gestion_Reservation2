# Gestion_Reservation2
## Description

Gestion_Reservation2 est une application Java développée dans le but de gérer un système de réservation.
Le projet permet de manipuler différentes entités liées aux réservations et de simuler certaines opérations telles que la création, la gestion et la consultation des données.

Ce projet a été réalisé dans un cadre pédagogique afin de pratiquer :

l’utilisation de Java

la gestion de la persistance des données avec JPA / Hibernate

l’architecture en couches

la manipulation d’une base de données relationnelle

Les données sont stockées dans une base de données et manipulées via les entités et les services de l’application.

---

# Technologies utilisées
    --> Java

    --> JPA (Java Persistence API)

    --> Hibernate

    --> H2 Database

    --> Maven

    --> IntelliJ IDEA

Ces technologies permettent de gérer la persistance des données et de structurer l'application de manière claire et maintenable.
